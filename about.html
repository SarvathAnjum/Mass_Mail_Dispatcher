What is Mass Mail Dispatcher?

Mass mailing is the act of sending the same email to a large group of people at one time. Mass Mail Dispatcher website allows its users to send mail to a large number of people easily with few simple steps. It is way more convenient than using any 3 party apps or softwares.

The 'SEND MAIL' button which is present on the home page directs user to the page where the mail can be sent. User needs to upload a csv file containing all the email addresses and enter some text into the subject and message box. On submitting, the mail will be send to all the valid and existing email addresses. The user can follow his/her email account to find out the status of the mail sent to each receipient.

On clicking the 'VALIDATE EMAIL' button, the user will be directed to an online python compiler where on successfully running the python script a structured list of valid and invalid email addresses will be displayed on the screen.

Who can use this website?

People involved in any kind of email marketing can use this website and make their work easy and quick.
